// INPUT //

// OUTPUT //

** 삼각형의 가장 긴 변은 나머지 두 변의 길이 합보다 작아야 삼각형이 생성될 수 있다. 즉, 두 변의 합이 가장 긴 변이랑 같아도 그것은 삼각형이 될 수 없다는 뜻이다. **
